Navigation Dropdown System Demo

This page demonstrates the modular navigation system for mindloft.org. Hover over the navigation items above to see dropdowns in action.

Features

  • Single Source of Truth: All navigation defined in js/nav-config.js
  • Hover Dropdowns: Desktop users see dropdowns on hover
  • Mobile-Friendly: Touch-optimized with click-to-expand
  • Active States: Current page automatically highlighted
  • Responsive: Hamburger menu on small screens
  • As-Code: Update navigation once, changes apply everywhere

Integration (3 steps)

To use this navigation system on any page:

Step 1: Include CSS/JS in <head> <link rel="stylesheet" href="/css/nav-dropdown.css"> <script src="/js/nav-config.js"></script> <script src="/js/nav-dropdown.js"></script>
Step 2: Add nav element to <body> <nav id="main-nav"></nav>
Step 3: Define CSS variables :root { --bg-deep: #050a14; --text-primary: #f1f5f9; --text-secondary: #94a3b8; --text-muted: #64748b; --accent-purple: #8b5cf6; --accent-cyan: #06b6d4; --accent-amber: #d4a543; }

That's it! The navigation will automatically populate and handle all interactions.

Adding New Pages

Edit js/nav-config.js to add pages to the navigation. See NAV-SYSTEM-README.md for full documentation.

Example: Add a new dropdown item { label: "Research", href: "#", dropdown: [ { label: "New Research Topic", href: "/research/new-topic/", description: "Description of the new page" } ] }
